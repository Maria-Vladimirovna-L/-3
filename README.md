# Movie Library (Личная кинотека)

## Автор
Сухов Матвей Витальевич

## Описание
Графическое приложение для ведения личной коллекции фильмов. Позволяет добавлять фильмы, просматривать их в виде таблицы, фильтровать по жанру и году выпуска. Все данные сохраняются локально в файл `movies.json`.

## Требования
- Python 3.8+
- Библиотека `tkinter` (встроена в стандартный дистрибутив Python)

## Установка и запуск
1. Клонируйте репозиторий:
   ```
   git clone https://github.com/yourname/movie-library.git
   ```
2. Перейдите в папку проекта.
3. Запустите приложение:
   ```
   python main.py
   ```

## Использование
1. Заполните поля: **Название**, **Жанр**, **Год выпуска**, **Рейтинг**.
2. Нажмите кнопку **«Добавить фильм»**.
3. Используйте фильтры **«Жанр»** и **«Год»** для поиска нужных фильмов.
4. Данные сохраняются автоматически при добавлении и загружаются при запуске программы.

## Примеры использования
*   **Добавление:** Название — «Начало», Жанр — «Фантастика», Год — 2010, Рейтинг — 8.8.
*   **Фильтрация:** Введите «2010» в поле «Год» и нажмите «Фильтровать», чтобы увидеть все фильмы за этот год.
*   **Очистка фильтра:** Нажмите «Сбросить фильтр», чтобы показать всю коллекцию.# -3
